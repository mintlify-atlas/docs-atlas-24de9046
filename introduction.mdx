# Welcome to Aphonos

Aphonos (formerly ALTERSHAPER) is a comprehensive Discord utility bot built specifically for ALTER EGO Wiki and Tower Defense Simulator Wiki communities. It combines powerful moderation tools with fun interactive commands to create an engaging server experience.

## What makes Aphonos special

Aphonos is purpose-built for wiki communities with features that go beyond standard moderation:

- **Wiki account linking**: Connect Discord users to their Fandom wiki accounts
- **Role synchronization**: Automatically assign roles based on wiki permissions
- **Comprehensive audit trail**: Track all moderation actions with unique identifiers
- **Message archiving**: Never lose context - deleted messages are automatically archived
- **Punishment reversal**: Use `/removesin` to reverse any moderation action by its ID

## Technology stack

Aphonos is built with modern technologies:

- **discord.js v14** - Powerful Discord API wrapper
- **TypeScript** - Type-safe development
- **Canvas** - Image generation for commands like /aura and /battle
- **Node.js** - Runtime environment
